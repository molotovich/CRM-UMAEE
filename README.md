# CRM Universitario Standalone

## Instalación
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python3 app.py`
4. ejecutar setup_mysql.sql para preparar la base de datos mysql
   
## Estructura
- `/backend`: Lógica de servidor y base de datos instance/crm.db.
- `/backend/static`: Archivos del frontend (HTML, CSS, JS).
